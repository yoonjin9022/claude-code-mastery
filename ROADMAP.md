# 개발자 웹 이력서 프로젝트 로드맵

## 프로젝트 개요

정적 웹 페이지 기반의 개인 개발자 이력서 사이트를 구축합니다.

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - 스타일링 및 애니메이션
- **JavaScript** - 인터랙션 및 동적 기능
- **Tailwind CSS** - 유틸리티 기반 스타일링

## 프로젝트 구조

```
/
├── index.html          # 메인 이력서 페이지
├── css/
│   └── style.css       # 커스텀 스타일
├── js/
│   └── main.js         # 인터랙션 로직
└── assets/
    └── images/         # 프로필 사진 등 이미지 리소스
```

## 이력서 섹션 구성

### 1. 헤더 / 히어로 영역
- 이름, 직함 (예: 풀스택 개발자)
- 한 줄 자기소개
- 프로필 사진
- SNS / GitHub / 이메일 링크

### 2. 자기소개 (About)
- 간단한 소개 문구 (3~4줄)
- 개발 철학 또는 관심 분야

### 3. 기술 스택 (Skills)
- 프론트엔드: HTML, CSS, JavaScript, React 등
- 백엔드: Node.js, Python 등
- 도구: Git, Docker, VS Code 등
- 아이콘 또는 프로그레스 바로 숙련도 표시

### 4. 경력 사항 (Experience)
- 회사명, 직책, 재직 기간
- 주요 업무 및 성과 요약

### 5. 프로젝트 (Projects)
- 프로젝트 카드 형태 (이미지 + 설명)
- 사용 기술 태그
- GitHub 링크 / 데모 링크

### 6. 학력 (Education)
- 학교명, 전공, 졸업 연도

### 7. 연락처 (Contact)
- 이메일, 전화번호
- 간단한 연락 폼 또는 링크

### 8. 푸터
- 저작권 표시
- 맨 위로 이동 버튼
